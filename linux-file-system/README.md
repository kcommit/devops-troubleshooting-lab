# Linux File System Study Notes - README

## Overview

This study notes file explains the **Linux file system from scratch to hero** in a simple and practical way.

It is designed for students, beginners, Linux administrators, and DevOps learners who want to understand how Linux organizes files, directories, devices, logs, permissions, and mounted storage.

## Main Study Notes File

```text
linux-file-system-from-scratch-to-hero-updated.md
```

## What You Will Learn

These notes cover:

- What the Linux file system is
- Why Linux needs a file system
- Linux root directory `/`
- One-line definitions of important Linux directories
- Absolute path and relative path
- File types in Linux
- Regular files and directories
- Symbolic links and hard links
- Inodes
- File permissions
- File ownership
- `chmod`, `chown`, and `chgrp`
- Disk partitions and mount points
- `/etc/fstab`
- Common file system types such as ext4, xfs, tmpfs, and NFS
- Important file system commands
- Hidden files
- File timestamps
- Special permissions
- File system troubleshooting
- Practical labs
- Interview questions
- Ubuntu vs RHEL notes
