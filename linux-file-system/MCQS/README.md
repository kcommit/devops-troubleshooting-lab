# Linux File System MCQ Quiz - README

## Overview

This HTML file contains a **25-question MCQ quiz** about the Linux file system.

It is designed for quick practice, revision, and student self-testing.

## File Included

```text
linux-file-system-25-mcq-quiz.html
```

## Features

- 25 multiple-choice questions
- Answer checking
- Score calculation
- 20-minute timer
- Show answers button
- Reset quiz button
- Short explanation for each answer
- Works directly in any web browser

## How to Use

1. Download the HTML file.
2. Double-click the file to open it in a browser.
3. Select one answer for each question.
4. Click **Submit Quiz** to check your score.
5. Read the short explanations for review.
6. Click **Reset Quiz** to practice again.

## Passing Score

```text
18 / 25
```

## Suggested Use

Use this quiz after studying the Linux file system notes.

Recommended topics to review before taking the quiz:

- `/`, `/home`, `/root`, `/etc`, `/bin`, `/sbin`
- `/var/log`, `/tmp`, `/dev`, `/boot`, `/mnt`, `/media`
- Absolute path and relative path
- File types
- Permissions
- Ownership
- Inodes
- Mount points
- `df`, `du`, `lsblk`, `chmod`, `chown`
